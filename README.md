# 🪽 Hermes – Lifestyle Rhythm App

**Hermes** is a daily rhythm web app designed to simplify decision-making, reduce overwhelm, and guide users into focused, intentional action.

> *Does life keep coming at you all at once? Are you overwhelmed by noise and choice?*
> Hermes helps cut through that chaos by guiding your mood, movement, and momentum—all in one place.

---

## ✨ Core Concept

Hermes acts as a **decision assistant + lifestyle companion**, helping users:

* Identify their current mood
* Receive curated music playlists
* Set and track movement goals
* Stay organized with a built-in daily planner

Instead of juggling multiple apps, Hermes creates a **single flow** that carries you from *feeling → action → progress*.

---

## 🚀 Features

### 🎭 Mood-Based Experience

* Users select their current mood (Focused, Calm, Pumped, etc)
* Hermes dynamically responds with a tailored experience

### 🎧 Smart Playlist System

* Embedded YouTube player powered by the YouTube IFrame API
* Features:

  * ▶️ Auto-play
  * ⏭ Next / ⏮ Previous controls
  * 🔁 Automatic progression when songs end
  * 🔢 Song counter (e.g. Song 1 / 2)

### 🏃 Movement Flow (Fitness Tracker)

* Guided prompts:

  * “Are we moving today?”
  * Type of movement (Walk, Run, Gym)
* Custom goal setting:

  * Steps, miles, or workout type
* Progress check-ins:

  * Tracks attempts
  * Encourages persistence
  * Provides feedback + completion state

### 📓 Daily Planner (Slide-Out Notebook)

* Expandable notebook UI
* Add, track, and complete tasks
* Features:

  * ✔️ Checkbox completion
  * ✏️ Persistent storage (localStorage)
  * 📂 Auto-load saved tasks

### 🌌 Immersive UI & Experience

* Animated loading screen:

  * “Hermes Lives!” text animation
  * Smooth fade transition into app
* Dynamic shimmering background (PS5-inspired)
* Floating particle effects for depth
* Neon cyber aesthetic with responsive UI

---

## 🧠 How It Works

Hermes follows a structured user flow:

1. **Mood Selection**
2. **Movement Prompt**
3. **Goal Setting**
4. **Session Start (Music + Tracking)**
5. **Progress Check-ins**
6. **Completion Feedback + Playlist Continuation**

This creates a **loop of engagement** designed to reinforce consistency and focus.
